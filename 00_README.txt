# LCPV_COVID_19
R script for generating waffle and lollipop plots for the manuscript titled, "Impact of COVID-19 on agriculture, biodiversity, and livelihoods in the Lake Cluster of Pokhara Valley, a Ramsar Site in Nepal" by Ram Bahadur Rana, Niranjan Pudasaini, Indra Paudel, Tejaswee Shiwakoti, and Sajal Ratna Sthapit.
R script authored by Sajal Sthapit. 
The plots were saved as pdfs and the session info with details about the packages and versions used are saved in the text file: sessionInfo.txt.
